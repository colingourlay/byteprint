Before you can install Column you need to provide your database settings:

  1. Open column/platform/database.py
  2. Complete each of the fields with your database settings:
    DATABASE_ENGINE
    Enter either mysql or postgresql
    DATABASE_NAME
    Enter the name of your database, i.e. column_database
    DATABASE_USER
    Enter the name of the user that has full read/write access to the database
    DATABASE_PASSWORD
    Enter the password used by the above user
    DATABASE_HOST
    If your database is running on another host, enter the address here, otherwise, leave blank
    DATABASE_PORT
    If your database is running on another host, enter the port number here; otherwise, leave blank
  3. Save and close column/platform/database.py
  4. Restart your server to load the new database settings

Once you have followed the steps above, you can continue installing Column.
